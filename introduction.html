What is an HTML Element?


An HTML element is defined by a start tag, some content, and an end tag. The HTML element is everything from the start tag to the end tag.

For exemple: <h1> as start tag, your content being Element content and </h1> the end tag.

HTML Attributes

It provides additional information about HTML elements.

Attributes usually come like name="value".

Commum attributes:

  • The href attribute used in <a> tag specifies the URL of the page the link goes to
  • The src attribute udes in <img> specifies the path to the image to be displayed.
  • The <style> attribute is used to add style to an element, such as color, font and more.
  • The title attribute defines extra information about an element and it's value will be displayed as a tooltip when you mouse over the element

The HTML Style Attribute

It has the following syntax:

<tagname style="property:value;">

Background color is a property from CSS and it defines the background color for an HTML element.

You can apply using as: <body style="background-color:red;">

Text Color property you can use like:

<h1 style="color:blue;">This text will be blue.</h1>

Text Alignment is a property that defines the horizontal text alignment for a HTML element.

Syntax:<p style="text-align:center;"> Centered Heading<p>

HTML Formatting Elements

They were designed to display special types of text.

  • <b> - Bold text
  • <strong> - Important text
  • <i> - Italic text
  • <em> - Emphasized text
  • <mark> - Marked text
  • <small> - Smaller text
  • <del> - Deleted text
  • <ins> - Inserted text
  • <sub> - Subscript text
  • <sup> - Superscript text

HTML Quotation and Citation Elements

  • <blockquote>- It element defines a section that is quoted from another source.
